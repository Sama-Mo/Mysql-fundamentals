# MySQL Tutorial & Practice

## 📌 Overview

This repository contains a collection of SQL scripts created while learning and practicing MySQL. It covers fundamental to intermediate SQL concepts with hands-on examples.

---

**## Topics Covered
**
* SELECT statements
* WHERE clause & filtering
* GROUP BY & ORDER BY
* HAVING vs WHERE
* Joins (INNER, LEFT, RIGHT)
* Subqueries
* Common Table Expressions (CTEs)
* Window Functions
* String Functions
* Case Statements
* Union operations
* Temporary Tables
* Stored Procedures
* Triggers and Events

---

**## Project Structure
**
Each file demonstrates a specific SQL concept:

* `Select Statement.sql` → Basic data retrieval
* `where clause.sql` → Filtering data
* `Group by and Order by.sql` → Aggregation and sorting
* `Having VS Where.sql` → Filtering aggregated data
* `Joins.sql` → Combining multiple tables
* `Subqueries.sql` → Nested queries
* `Common Table Expressions CTEs.sql` → Advanced query structuring
* `Window Functions.sql` → Analytical functions
* `String Functions.sql` → Text manipulation
* `Case Statements.sql` → Conditional logic
* `Union.sql` → Combining query results
* `Temporary Tables.sql` → Temporary data storage
* `Stored Procedures.sql` → Reusable SQL logic
* `Triggers and Events.sql` → Automation in databases
* `Limit and Aliasing.sql` → Result control and readability

**## Project Goal
**
To build a strong foundation in SQL and understand how to query, manipulate, and analyze relational data efficiently.
